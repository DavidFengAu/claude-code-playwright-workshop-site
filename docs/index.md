## Welcome to the Workshop

This hands-on workshop teaches you how to **leverage AI to automate web testing** using **Claude Code** and **Playwright**. Whether you're a QA engineer, developer, product manager, or business analyst, you'll learn how to create robust automated tests without deep programming knowledge.

### What You'll Need

Before starting the workshop, ensure you have:

1. **Claude Code** - Your AI coding assistant ([Get Claude Code](https://www.claude.com/product/claude-code))
2. **Node.js** - Version 18 or higher ([Download Node.js](https://nodejs.org/))
3. **A Code Editor** - VS Code, WebStorm, or your preferred IDE
4. **Internet Connection** - For installing packages and accessing the demo application

::: warning No Coding Experience Required
This workshop is designed for all skill levels. If you can describe what you want to test in plain English, Claude Code will help you create the automation code.
:::

### Workshop Format

Each workshop follows a consistent structure:

1. **Concept Introduction** - Learn the key concepts with clear explanations and examples
2. **Hands-On Exercise** - Practice what you've learned with guided exercises
3. **Solution Review** - Compare your work with the provided solutions
